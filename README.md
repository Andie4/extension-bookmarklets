# Extension de bookmarklets d'accessibilité 

L'extension regroupe les bookmarklets disponible sur A11Y.tools.com <a href="https://a11y-tools.com/bookmarklets/">A11Y Tools</a>

## Prise en main :
Pour récupérer l'extension il faut : 
1. Télécharger le dossier extensions-bookmarklets
2. Entrez cette URl dans le navigateur Firefox "about:debugging"
3. Cliquez sur "Ce firefox"
4. Cliquez sur "Charger un module complémentaire temporaire..."
5. Sélectionnez le fichier "manifest.json"
6. Vous pouvez dès à présent voir l'icon de l'extension et cliquer dessus pour l'utiliser. 


## Environnement de travail :
Cette extension est uniquement fonctionnelles sur Firefox.
